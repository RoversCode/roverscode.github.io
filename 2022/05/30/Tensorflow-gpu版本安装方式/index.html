Tensorflow-gpu版本安装方式

Posted on 2022-05-30 Edited on 2022-07-02 In Tensorflow , GetStart Views: Disqus: Symbols count in article: 402 Reading time ≈ 1 mins.

conda方式

确定自己的显卡是否支持CPU

• CUDA® 架构为 3.5、5.0、6.0、7.0、7.5、8.0 或更高的 NVIDIA® GPU 卡。请参阅支持 CUDA® 的 GPU 卡列表。

查看自己的显卡驱动版本支持的cudatoolkit

  • win+R cmd 进入终端

  • 输入命令nvidia-smi

CUDA Version表示支持的最高版本，一般来说是向下兼容的

查看自己conda源有的tensorflow-gpu包的版本

  • conda search tensorflow

在上一步的基础上，查看自己的Python版本是否满足对应的tf-gpu版本

在Tensorflow官网在 Windows 环境中从源代码构建  |  TensorFlow寻找

根据上面所得的信息，查看自己要gpu版本，用conda进行安装

conda install tensorflor-gpu==2.5.0

conda会自动找到相应的cudatoolkit和cudann版本！
